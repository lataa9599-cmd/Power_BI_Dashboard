This project is a Power BI dashboard built using Excel as the data source. 
It is designed based on real business reporting requirements to track and visualize key metrics.

#Overview

The dashboard provides insights by transforming and visualizing data from Excel into an interactive Power BI report.

#Data source: Excel
Data is cleaned and transformed inside Power BI
Dashboard presents key business metrics in a clear and visual format

#Impact
Reduced manual reporting effort
Improved data visibility and decision-making
Simplified reporting process using automation within Power BI

#Feature

Interactive visuals and filters
Dynamic data updates from Excel
Clean and structured dashboard design
Business-focused KPIs and insights
